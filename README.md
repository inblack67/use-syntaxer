## use-syntaxer

Enter your markdown code and get back a perfect render with code syntax highlighting.

## Usage

```jsx
import { RenderMarkdown } from 'use-syntaxer';

const index = () => {
  return (
    <div>
      <RenderMarkdown code={code} />
    </div>
  );
};
```
